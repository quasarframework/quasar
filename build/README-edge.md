# Quasar Framework - "Edge" / "Nightly" Build

Use "edge" / "nightly" build for bleeding edge of **current major version** of Quasar. No breaking changes.

``` bash
npm i --save git+https://git@github.com/quasarframework/quasar-edge.git
```
