# Quasar AE <%= name %> Playground (quasar-cli-webpack)

## Install the dependencies

```bash
pnpm install
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)

```bash
pnpm dev
```

### Lint the files

```bash
pnpm lint
```

### Format the files

```bash
pnpm format
```

### Build the app for production

```bash
pnpm build
```
